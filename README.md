## 安装
```
1. composer install
2. 配置redis连接
3. 配置config/autoload/filestore.php的cos配置

```
## 采集

```
php artisan bili:img 哔哩哔哩up主的id

例如: php artisan bili:img 424692353

即采集: https://space.bilibili.com/424692353/album
```


## 运行
```
php bin/hyperf start
```
